This sample code and utility scripts may be used to test goto functionality 
in Java.

Prerequesites:

1. Java 8 installed.
2. Eclipse with Java 8 support or similar IDE
3. Download openjdk (requires mercurial (hg))
4. Set up softlinks to openjdk and javac source
   openjdk8 -> /Users/marska/Google Drive/Jobb/faglig/goto/workspace/openjdk8
   javac -> openjdk/langtools/src/share/classes

5. For help with required changes to OpenJDK contact martin@skarsaune.net
